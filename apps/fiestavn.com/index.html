About

FiestaVN is an open-source Dart and Flutter implementation of the Ren'Py visual novel engine, built by Fiesta Berry Games. Its first app, RenFly, plays Ren'Py games wherever Flutter runs. The parser, engine core, and Flutter runtime are published as standalone packages you can build on and are all licensed MIT.

RenSpine: RenFly and Spine for animated visual novels

RenSpine combines the RenFly Ren'Py player with Esoteric Software's Spine, so visual novels can use skeletal-animated characters — with skins and animation mixing — instead of static sprites. Try the live demo, or build on the packages/renpy_spine package in the FiestaVN monorepo.
